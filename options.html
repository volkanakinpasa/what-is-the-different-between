Templates

Default search engines is Google. If you want to use another one please copy and paste one of the links below.
  • http://www.google.com/search?q=
  • https://search.yahoo.com/search?p=
  • https://www.bing.com/search?q=
  • other search engines? please create a issue https://github.com/volkanakinpasa/what-is-the-different-between/issues




How to use

  • Type 'diff' on address bar
  • Press space or tab. You will see address bar is ready for DIFF search
  • Type what you would like to search. example: 'humans and animals'
  • Press enter
  • It will search on Google as 'What is the difference between human and animal'
